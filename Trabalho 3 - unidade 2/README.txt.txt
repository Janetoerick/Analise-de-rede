Neste trabalho é feita várias análises direcionadas à grafos, algumas dessas análises são sobre a Excentricidade, diâmetro,
periféricos, entre várias outras que possíbilitam ter uma ideia enclarecedora sobre a rede analisada.

Para aplicar todas as análises foi utilizado um DataFrame contendo playlists do Spotify, no qual em cada playlist há
um número diferente de músicas, com músicas que variam dês de músicas internacionais como também músicas nacionais. A partir
desse DataFrame foi construido um grafo não direcionado, onde cada nó representa o nome do artista referente as músicas das
playlists e as arestas no grafo representam a ligação de cada artista com outro através das playlists.
